Sintaxis Básica
1. Seleccionar Nodos
Elemento específico:
Selecciona un nodo por su nombre.

xpath
Copiar código
/libro/titulo
Devuelve el nodo <titulo>.
Todos los nodos de un tipo:
Usa // para seleccionar todos los nodos en cualquier lugar del documento.

xpath
Copiar código
//titulo
Encuentra todos los nodos <titulo>.
2. Predicados
Seleccionar nodos según una condición:

xpath
Copiar código
//libro[autor="Juan Perez"]
Encuentra <libro> con un autor llamado "Juan Perez".
Seleccionar por posición:

xpath
Copiar código
/libro/capitulo[1]
Devuelve el primer nodo <capitulo>.
3. Atributos
Seleccionar nodos por atributo:

xpath
Copiar código
//libro[@categoria="educación"]
Encuentra <libro> cuyo atributo categoria sea "educación".
Obtener el valor de un atributo:

xpath
Copiar código
//libro/@categoria
Devuelve el valor del atributo categoria de <libro>.
4. Funciones Comunes
text(): Selecciona el texto dentro de un nodo.

xpath
Copiar código
//titulo/text()
Devuelve "Aprender XPath".
contains(): Verifica si un nodo contiene un texto.

xpath
Copiar código
//titulo[contains(text(), "XPath")]
Encuentra nodos <titulo> cuyo texto contenga "XPath".
starts-with(): Verifica si un texto comienza con una cadena específica.

xpath
Copiar código
//titulo[starts-with(text(), "Aprender")]
5. Operadores
Relacionales: =, !=, <, <=, >, >=

xpath
Copiar código
//libro[precio > 20]
Encuentra libros con un precio mayor a 20.
Lógicos: and, or, not()

xpath
Copiar código
//libro[autor="Juan Perez" and precio > 20]
Encuentra libros de Juan Perez con un precio mayor a 20.
Ejemplo Completo
XML
xml
Copiar código
<biblioteca>
    <libro categoria="educación">
        <titulo>Aprender XPath</titulo>
        <autor>Juan Perez</autor>
        <precio>25</precio>
    </libro>
    <libro categoria="novela">
        <titulo>El camino</titulo>
        <autor>Carlos Ruiz</autor>
        <precio>15</precio>
    </libro>
</biblioteca>
Consultas XPath
Seleccionar todos los libros:

xpath
Copiar código
/biblioteca/libro
Resultado: Devuelve los dos nodos <libro>.

Seleccionar el título del primer libro:

xpath
Copiar código
/biblioteca/libro[1]/titulo
Resultado: <titulo>Aprender XPath</titulo>

Encontrar libros cuyo precio sea mayor a 20:

xpath
Copiar código
//libro[precio > 20]
Resultado: <libro categoria="educación">...</libro>

Obtener el valor del atributo categoria de todos los libros:

xpath
Copiar código
//libro/@categoria
Resultado: "educación", "novela"
